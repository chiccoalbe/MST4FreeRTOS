Detailed Description

Header file for the standard model adaptation library of FreeRTOS.

Author
Enrico Alberti
Date
2024

This file contains definitions and function prototypes for the Model-Specific Task (MST) scheduler library, which extends FreeRTOS to include additional scheduling policies and support for periodic and sporadic tasks.
